Features

Data Catalog Management


Automatically obtain datasource metadata and construct enterprise-level data catalog; Regularly monitor data schema changes.

Data Quality Monitor


Multiple data quality check rules are built in the platform, which can be used out of the box; Provide alert message management to help you better grasp the quality situation.

Plug-in Design


With plug-in design as the core, the platform supports plug-in design of datasource, rule, notification, error data storage and registry.

Data Profile


Data profile enables you to understand the situation of each data set faster and more easily, so that you have a clear idea.

Multiple Mode


Not only provides a platform to help you observe data, but also supports local mode integration into the workflow of the scheduling system.

Easy Deploy & HA


Less dependent and easy to deploy; Supports horizontal expansion and automatic fault tolerance.
